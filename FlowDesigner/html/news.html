22/10/2004
FlowDesigner 0.8.2 released! Please see download section to download sources or binaries.

*** Added features ***
- Now able to create notes (GUINotes) in the user interface. Notes can be added anywhere in any Network like post-it notes. Still need some work, but working properly. Notes can be moved anywhere, but they do not always appear at the good position when creating them.
- Now able to import / export already existing networks. Next step is to have the ability to modify imported networks directly.

*** Fixed features ****
- Now display the network type (iterator, subnet) properly in the Network tab(s)
- Node Parameters dialog working properly. Still need to bring back focus when dialog is minimized.

*** Help Needed ***
- We need help to continue our Octave toolkit. Maybe you can contribute?
- We need help to create RPM and deb packages for the most popular Linux distributions. Please contact us if you have made RPMs and / or Debian packages so we can make them available for everybody.

Please report bugs at theFlowDesigner SourceForge Page.
25/06/2004
FlowDesigner 0.8.1 released! Please see download section to download sources or binaries. This release includes new URL streams that can be specified by users at run time by using : tcp:host:port, file:filename, etc. Added automatic type conversion when using RCPtr = ObjectRef. Fixed parameters passing at run-time from document parameters. Now working with MARIE. Fixed String serialization. Added Vector conversion with toInt, toFloat nodes. Removed ownership on sockets, which causes exceptions on some systems. Work around what seems to be gnome header bugs. Reordered #include"s so that gnome.h gets included after other headers. Octave plugin now parses octave file and executes properly, still need more work on this toolbox. Added a lot of operators on basic types : add, sub, mul, div, concat. Please report bugs at the FlowDesigner SourceForge Page.
