The source is also available via anonymous CVS:

export CVSROOT=:pserver:anonymous@cvs.on.openprojects.net:/cvs/freespeech
or (for csh users) setenv CVSROOT :pserver:anonymous@cvs.on.openprojects.net:/cvs/freespeech
cvs login  (the password is return)
cvs checkout FreeSpeech
