FreeSpeech

FreeSpeech a free (GPL) speech recognition project for Linux (or any other UNIX flavor). It will be designed so that it can be easily integrated into any application or window manager as well as the KDEand gnomedesktop environments.

For those who are interrested, you can download a daily tarball of the CVS here.
