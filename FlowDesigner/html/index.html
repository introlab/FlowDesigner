SpeechInput

SpeechInput a free (GPL) speech recognition project for Linux (or any other UNIX flavor). It will be designed so that it can be easily integrated into any application or window manager as well as the KDE and gnome desktop environments. We are still in the planning phase and very little code has been written at this point. For those who are interrested, this code is available here.
