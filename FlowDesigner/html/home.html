FlowDesigner is a free (GPL/LGPL) "data flow oriented" development environment. It can be use to build complex applications by combining small, reusable building blocks. In some way, it has similarities with Simulink and LabView, although it is not designed (and far) to be a "clone" of any of them.

FlowDesigner features a GUI that allows rapid application development and features a visual debugger. Although FlowDesigner can be used as a rapid prototyping tool, it can still be used for building real-time applications, such as audio effects processing. Since FlowDesigner is not really an "interpreted language", it can be quite fast.

It is written in C++ and features a plugin mechanism that allows plugins/toolboxes to be easiliy added. FlowDesigner currently includes the following toolboxes:

  • Signal processing
  • Audio processing (DSP)
  • Vector quantization
  • Neural network
  • Fuzzy logic
  • Real-time audio effect processing (early stage)
  • Linear algebra using LAPACK (early stage)
  • Robotics
